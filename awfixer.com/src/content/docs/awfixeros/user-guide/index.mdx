# AWFixerOS User Guide

This guide covers everything you need to know to use AWFixerOS effectively in your daily workflow.

## What You'll Learn

- Navigate the desktop environment
- Manage files and folders
- Install and manage software
- Configure system settings
- Customize your workspace

## Chapters

1. **[Desktop Environment](/docs/awfixeros/user-guide/desktop-environment)** - Understanding the AWFixer desktop
2. **[File Management](/docs/awfixeros/user-guide/file-management)** - Working with files and folders
3. **Software Installation** - Installing applications (coming soon)
4. **System Settings** - Configuring your system (coming soon)
5. **Networking** - Connecting to networks (coming soon)

## Prerequisites

Before starting this guide, you should have:

- AWFixerOS installed on your system
- Completed the initial setup wizard
- Basic familiarity with computer usage

## Getting Help

If you get stuck:

1. Use the built-in help: `awfix help <command>`
2. Search the documentation
3. Ask in our [Discord community](https://discord.gg/awfixer)
