# AWFixerOS Documentation

AWFixerOS is a modern, security-focused Linux distribution designed for developers, system administrators, and power users.

## Why AWFixerOS?

### Security First

- Hardened kernel with security patches
- Mandatory access controls (MAC)
- Full disk encryption by default
- Regular security updates

### Developer Experience

- Pre-configured development tools
- Container-native workflow
- Modern shell experience
- Extensive language support

### Performance

- Optimized for modern hardware
- Fast boot times
- Efficient resource usage
- Rolling release model

## Quick Reference

### Common Commands

```bash
# System update
awfix update && awfix upgrade

# Package management
awfix install <package>
awfix remove <package>
awfix search <query>

# System info
awfix system info
awfix system health

# Service management
awfix service start <service>
awfix service status <service>
```

### Key Locations

| Path | Description |
|------|-------------|
| `~/.config/awfixer/` | User configuration |
| `/etc/awfixer/` | System configuration |
| `/var/log/awfixer/` | System logs |
| `/opt/awfixer/` | Optional packages |
